PlantUML 출력 (B-out)

flowcast는 같은 흐름도 JSON을 scripts/plantuml_export.py로 .puml 텍스트로도 뽑는다 (stdlib만, 좌표 미사용). 아래는 HTML 갤러리와 동일한 합성 예제를 PlantUML로 export → plantuml -tsvg로 렌더한 결과다. 시나리오마다 다이어그램 1개(@startuml 블록 1개).

노드 id는 화살표 문법과 충돌하지 않도록 별칭으로 정규화되고(예: fw-edge → fw_edge), 표시명은 원문 그대로 유지된다.

직접 만들기

python3 scripts/plantuml_export.py {view.json} -o out.puml   # JSON → .puml (stdlib)
plantuml -tsvg out.puml                                     # .puml → SVG (graphviz dot)
